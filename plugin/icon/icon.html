アイコンの記述方法

spanタグでhtml内に記述

<span class="ui-icon ui-icon-power"></span>

【設定内容】.ui-icon-{種類}-{小種別}-{方向}

  • 種類：icon type
  • 小種別：icon sub description
  • 方向：direction
　　※spanタグの間にテキストを記述しても表示されない 　　※subやdirectionの設定がない場合は無視される

subやdirectionの設定例

種別1

  • 　ui-icon-arrow-1-n
  • 　ui-icon-arrow-1-ne
  • 　ui-icon-arrow-1-e
  • 　ui-icon-arrow-1-s
  • 　ui-icon-arrow-1-w
  • 　ui-icon-arrow
  • 　設定なし or 設定ミス
  • 　ui-icon-arrow-1-en

種別2

  • 　ui-icon-arrow-2-n
  • 　ui-icon-arrow-2-ne
  • 　ui-icon-arrow-2-e
  • 　ui-icon-arrow-2-s
  • 　ui-icon-arrow-2-w
  • 　ui-icon-arrow-2-en
