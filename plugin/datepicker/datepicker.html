DatePicker ~jQuery Plugin~

参考サイト
jQuery公式 Datepicker widget

(1)Default

出力値 mm/dd/yy

Date: 

    ①inputタグ、id/classを設定（class="datepicker"）
    ②「.datepicker()」を使う
    ※cssの設定なし
    ※defaultは「月/日/年」

    $( function() {
        $( ".datepicker" ).datepicker();
    } );

(2)Format変更

①出力値 yyyy/mm/dd(曜日)

Date: 

    ※引数で「dataFormat」を設定
    ※yy = 2020 (4桁)
    ※DD = 曜日（English）
    $( ".datepicker2" ).datepicker({
        dateFormat: 'yy/mm/dd (DD)'
    });

②カレンダー見出し 〇年〇月
　 曜日を漢字＋任意にカスタマイズ

Date: 

    ※「yeatSuffix」: カレンダー上部 西暦4桁の後ろに表示する文字
    ※「showMonthAfterYear」:カレンダー上部 並びを年→月にする
    ※「monthNames」: カレンダー上部 月の表示を指定（任意の文字列）
    ※「dayNameMin」: カレンダー中部 曜日の表示を指定（任意の文字列）

    $( ".datepicker3" ).datepicker({
        yearSuffix: "年",
        showMonthAfterYear: true,
        monthNames:['1月', '2月', '3月', 'AAA', '5月', 'BBB', '7月', '8月', '9月', '10月', '11月', '12月'],
        dayNamesMin:['日', 'CCC', '火', '水', 'DDD', '金', '土']
    });

③出力値 yyyy年m月d日(曜日)

Date: 

    ※「dateFormat」で表示を'yy年m月d日 (DD)にする
    ※「dayNames」で DDの英語標記を、指定した文字列に変更する

    $( ".datepicker4" ).datepicker({
        dateFormat: 'yy年m月d日 (DD)',
        dayNames: ['日', '月', '火', '水', '木', '金', '土'],
    });

④カレンダーの表示を30日後に設定

Date: 

    ※「defaultDate：±数値」で設定
    ※ハイライトは今日から動かせない
    $( ".datepicker5" ).datepicker({
        defaultDate: +30
    });

⑤日本仕様(カレンダー1年後表示)

Date: 

    // カレンダーの設定
    yearSuffix: "年",
    showMonthAfterYear: true,
    monthNames:['1月', '2月', '3月', '4月', '5月', '6月', '7月', '8月', '9月', '10月', '11月', '12月'],
    dayNamesMin:['日', '月', '火', '水', '木', '金', '土'],
    firstDay: 1,

    // 出力の設定
    dateFormat: 'yy年m月d日 (DD)',
    dayNames: ['日', '月', '火', '水', '木', '金', '土'],

    // 初期表示の設定
    defaultDate: '+2y'　*数値 or クオテーション＆m or yで指定。
    （'+1y' ＝ '+12m' ≒ +365）

⑥最大 選択可能日付の設定、アイコンなし

Date: 

    ※「maxDate」最大選択可能日付
    ※「hideIfNoPrevNext:true」最大/最小日付以降の次へアイコン非表示
    $( ".datepicker7" ).datepicker({
        maxDate:'+5w',
        hideIfNoPrevNext:true
    });

⑦最小 選択可能日付の設定

Date: 

    ※「minDate」最小選択可能日付
    $( ".datepicker8" ).datepicker({
        minDate:'+1w', 　//一週間後以降
    });

➈最大(今日)、最小(2ヶ月前)、アイコンなし

Date: 

    ※「new Date()」今日(最新の日付)
    $( ".datepicker9" ).datepicker({
        maxDate:new Date(),
        minDate: '-2m',
        hideIfNoPrevNext:true
    });

➉土曜始まり

Date: 

    ※「firstDay:0~6」
    ※デフォルト０＝日曜日
    ※７＝日曜日になる
    ※1：月、2：火、3：水、4：木、5：金、6：土
    $( ".datepicker10" ).datepicker({
        firstDay: 6
    });
