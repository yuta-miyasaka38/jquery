Autocomplete ~jQuery Plugin~

(1)Default

Resizable

AAA
    ※divタグ(id="resizable")
    ※.resizable() ・・・拡大縮小
    ※デフォルト、最大サイズ、最小サイズの設定
    【JS】
    $( "#resizable" ).resizable();

    【CSS】
    #resizable {
        height: 120px; width: 200px;
        max-width: 90%; max-height: 500px;
        min-width: 80px; min-height: 80px;
    }

(2)画像resizable

画像を直接resizeableにした場合

divタグの中にいれた場合

    ・画像に直で設定するとアイコンが見えない → paddingの設定で解決
    ・親要素に設定する場合は画像に max-height, max-width　を設定する
    　　- 設定しないとオリジナルサイズではみ出す

(3)Animation

Animate

    ※resizableの引数に、「animate: true」を設定
    $( ".resizableA" ).resizable({
        animate: true
    });


    ※cssでhelperを設定（グレーのドット枠線）
      -helperがないとわかりにくい 
    .ui-resizable-helper { border: 1px dotted gray; }

(4)Constrain Container(決められた枠内でのみ操作可能)

コンテナ

Resizable要素

    ※resizableの引数に、「containtment: ".container"」を設定
    ※".container"は親要素

    $( ".resizableC" ).resizable({
        containtment: ".container"
    });


    ------css--------
    ※div(class="container")の大きさはcssで指定
    .container {
        width: 70%;
        height: 300px;
    }

(5)Animate & Container

コンテナ

Resizable要素

    ※resizableの引数に、「animate」と「container」を設定
    ※containerとすると、上の枠を引き継いでしまい高さ変更ができない → container2で別に定義
    $( ".resizableAC").resizable({
        animate: true,
        containment: ".container2",
      });

(6)Reserve Aspect Ratio

Preserve aspect ratio

    ※resizableの引数で、aspect比を指定
    $( ".resizableAspect" ).resizable({
        aspectRatio: 16 / 9
      });

(7)Snap to grid(gridに合わせて拡大)

Grid

    ※resizableの引数で、gridと幅を指定
    $( ".resizableG" ).resizable({
        grid: 50
    });
